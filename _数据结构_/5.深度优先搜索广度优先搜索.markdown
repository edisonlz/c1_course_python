##### 深度优先搜索(DFS) Depth First Search

tree = ["A","B","C","D","E","F","G"]

        A
      /   \
     B     C
    /\     /\
   D  E   F  G

这里采用-后序遍历, DEBFGCA


 ##### 如何实现
 深度优先搜索使用[栈]来实现 LIFO
1.遍历左子树
2.遍历右子树
3.遍历父节点

---------------
